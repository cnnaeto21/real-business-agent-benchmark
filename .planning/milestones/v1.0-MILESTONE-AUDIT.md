### Cross-Phase Wiring Issues

**1. DASH-04 — Metadata not rendered in ScoresTable** (minor)
- `web/lib/data.ts` correctly populates `run_date`, `harness_version`, `temperature` on each `RunResult`
- `web/components/ScoresTable.tsx` renders only `model`, `harness`, `composite_score`, `cost_usd`, `latency_ms`
- Three fields required by DASH-04 are loaded but invisible to dashboard visitors

**2. DASH-05 — RunItYourself display/copy mismatch** (minor)
- Visible code block: `anthropic/claude-3-5-sonnet` (stale)
- CopyButton copies: `anthropic/claude-sonnet-4-6` (correct)
- User reads one model, pastes another

**3. EVAL-02 — judge_temperature spec field not wired** (informational)
- `harness.yaml eval.judge_temperature: 0` parsed into `spec` but `callJudge()` hardcodes `temperature: 0`
- Behavior correct by coincidence; future rubric changes to judge temperature would silently fail

**4. HRNS-02 — description field dead in TypeScript type** (informational)
- `harness.yaml description:` field exists in all three YAMLs
- `HarnessSpec` type has no `description` field; field silently ignored
